# Public Repository Notice

This is the **PUBLIC RELEASE** version of VitalFlow-Radar for hackathon submission and open-source distribution.

## ⚠️ Important Notes

### Configuration Required
Before running this project, you need to set up your own credentials:

1. **Copy the environment template:**
   ```bash
   cp .env.example .env
   ```

2. **Fill in your credentials:**
   - Confluent Cloud: Sign up at https://www.confluent.io/confluent-cloud/tryfree/
   - Google Cloud: Create project at https://console.cloud.google.com/
   - Follow the detailed setup guide in `QUICK_START.md`
